# GymNow: Edge-AI Workout Generator

GymNow is a mobile application that brings deep learning directly to the user's device. It generates highly customized, dynamic workout routines using a trained neural network deployed entirely on the edge.

## ⚙️ How It Works
* **Model Training:** A custom deep learning model was trained on extensive fitness datasets to map user physical metrics and goals to optimal exercise combinations.
* **Edge Deployment:** The heavy model was converted into **TensorFlow Lite (.tflite)** format, allowing it to run locally on mobile devices with zero latency and no server dependency.
* **Mobile Interface:** The AI engine is wrapped in a smooth, cross-platform **Flutter** application.

## 🛠️ Tech Stack
* **AI/ML:** Python, TensorFlow, TensorFlow Lite
* **Mobile Development:** Flutter, Dart
* **Core Concepts:** Edge AI, Dataset Training, Mobile AI Integration
